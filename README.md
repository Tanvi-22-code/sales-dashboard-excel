## Overview
This project is an interactive Sales Dashboard created using Advanced Microsoft Excel.  
It analyzes sales performance across multiple cities and sales executives, helping track targets, achievements, and performance gaps.

The dashboard is designed to support data-driven decision-making for sales teams and managers.

## Objectives
- Analyze total sales by sales executive
- Measure target achievement percentage
- Identify gap between actual sales and targets
- Compare performance across different cities

## Tools & Features Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers for dynamic city-wise filtering
- KPI Metrics (Target Hit % and Away From Target %)
- Data cleaning, formatting, and visualization

## Dashboard Highlights
- City-wise interactive filtering using slicers
- Sales executive performance comparison
- Target vs actual sales analysis
- Visual insights using bar, pie, and line charts

## Key Insights
- Identified top-performing and underperforming sales executives
- Highlighted cities with higher target achievement
- Detected gaps between targets and actual sales

## Use Case
This dashboard can be used by:
- Sales Managers to monitor team performance
- Management to evaluate target achievement
- Analysts to quickly identify performance trends

## Project Type
Advanced Excel | Data Analysis | Dashboarding
